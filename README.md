# tic-tac-toe

Simple Naughts & Crosses Game for 2 x Players Playing on the Same Computer

Features: 
  - Animated header
  - Buttons: 
      - New Game
      - Clear Score
      - Edit Names
          - Opens Forms to Fill in Names
  - Player Scoreboards
      - Name (Player 1 / 2 by default)
      - Maker:  ❌ / ⭕
      - Score
  - Game Results:
      - Displays next player to move / winner / game tied
